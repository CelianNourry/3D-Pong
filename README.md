## Description
This is a 3D Pong game with a beach theme. A ball is launched and must be caught by the player and a bot opponent who pass it back and forth.

## Features
- 3D graphics with shading and textures  
- Player movement controlled by the mouse, and an AI opponent that automatically tries to catch the ball  
- The ball’s speed increases over time  
- Opponent scoring system: if the player fails to catch the ball 3 times, the game ends  

## Installation
1. Clone the repository  
2. Install the required libraries  
3. Compile the code using the provided makefile  
4. Run the executable file (ellule)
